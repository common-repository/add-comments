Add multiple comments at once to a post or page with whatever author information you want..

== Description ==
Adds an options page that has a dropdown menu of all your posts and pages. You select what author information you want to appear, and paste a block of text into the comment section. Use `%*%` to break the text into seperate comments.

== Installation ==
1. Download the latest zip file and extract the `add-comments` directory.
2. Upload this directory inside your `/wp-content/plugins/` directory.
3. Activate 'Literal Shortcode' on the 'Plugins' menu in WordPress.
4. Browse to the `Add Comments` page on the Options menu.

== Frequently Asked Questions ==
= How do I add comments? =
Browse to the `Add Comments` page on the Options menu and put your author info into the form. Paste your comments into the box seperated by `%*%`. Click Add Comments.
